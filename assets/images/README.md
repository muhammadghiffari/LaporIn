# Logo LaporIn

Silakan tambahkan file gambar logo LaporIn dengan nama `laporin-logo.png` di folder ini.

## Spesifikasi Logo

- **Nama file**: `laporin-logo.png`
- **Format**: PNG (dengan transparansi jika diperlukan)
- **Ukuran recommended**: 
  - Width: 800-1200px
  - Height: Auto (sesuai aspect ratio)
  - Atau square: 1024x1024px

## Deskripsi Logo

Logo LaporIn menampilkan:
- Teks "LAPORIN" dengan font modern dan gradient cyan/blue
- Huruf 'O' diganti dengan square icon yang memiliki simbol di dalamnya
- Background: Hitam atau transparan
- Warna: Gradient dari light blue/cyan ke greener cyan

## Cara Menambahkan

1. Simpan file gambar logo dengan nama `laporin-logo.png`
2. Letakkan di folder ini: `assets/images/laporin-logo.png`
3. README.md akan otomatis menampilkan logo

## Alternatif

Jika ingin menggunakan URL gambar (dari GitHub atau hosting lain), edit README.md dan ganti path dengan URL:
```markdown
![LaporIn Logo](https://your-image-url.com/laporin-logo.png)
```
