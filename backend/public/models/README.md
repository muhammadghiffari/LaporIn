# Face Recognition Models

Folder ini berisi model weights untuk face-api.js.

## Setup

Jalankan script download:

```bash
cd backend
npm run download:face-models
```

Atau download manual dari:
https://github.com/justadudewhohacks/face-api.js-models/tree/master/weights

## Files Required

1. `tiny_face_detector_model-weights_manifest.json`
2. `tiny_face_detector_model-shard1`
3. `face_landmark_68_model-weights_manifest.json`
4. `face_landmark_68_model-shard1`
5. `face_recognition_model-weights_manifest.json`
6. `face_recognition_model-shard1`
7. `face_recognition_model-shard2`

## Note

Models ini tidak sensitif dan bisa di-commit ke git repository.
